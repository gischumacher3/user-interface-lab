# 🚗 Sistema de Reservas de Viagens

## 📌 Sobre o Projeto

Sistema desenvolvido em Java com foco em Programação Orientada a Objetos (POO), simulando uma plataforma de reservas de viagens com gerenciamento de usuários, veículos e motoristas.

O sistema permite o cadastro de usuários, registro de veículos e realização de reservas com validação de disponibilidade e cálculo automático do valor da viagem.

<br>

## 🎯 Objetivos

- Simular um sistema real de reservas de viagens.  
- Aplicar conceitos de Programação Orientada a Objetos.  
- Desenvolver lógica de negócio com regras e validações.  
- Organizar o código de forma estruturada e reutilizável.  
- Trabalhar relacionamento entre classes e objetos.

<br>

## 🧩 Funcionalidades

- Cadastro de usuários (passageiros e motoristas).  
- Registro de veículos com informações de modelo e valor da viagem.  
- Verificação de disponibilidade do veículo.  
- Realização de reservas de viagem.  
- Cálculo automático do custo da viagem.  
- Atualização do status do veículo após reserva.  
- Exibição de informações do passageiro e da reserva.

<br>

## 🏗 Estrutura do Projeto

Sistema de Reservas de Viagens

- Usuario → Representa os usuários do sistema (nome, e-mail, senha).  
- Veiculo → Representa os veículos disponíveis para viagem.  
- ReservarViagem → Responsável pela lógica de reserva e validações.  
- Main → Classe principal responsável por executar a simulação do sistema.

<br>

## ⚙️ Regras de Negócio

- Um veículo só pode ser reservado se estiver disponível.  
- Após a reserva, o veículo é marcado como indisponível.  
- O valor da viagem é calculado automaticamente.  
- Cada reserva está associada a um usuário e um veículo.

<br>

## 🚀 Como Executar

1. Clone o repositório:
   `git clone https://github.com/SEU_USUARIO/sistema-reservas-viagens.git`

2. Abra o projeto em uma IDE Java (IntelliJ, Eclipse ou VS Code).

3. Execute a classe `Main`.

4. A simulação do sistema será iniciada no console.
